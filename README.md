# gt-security-msr
A glamorous toolkit development about security

## Installation

```st
Metacello new
	repository: 'github://jesus-acev/gt-security-msr:main/src';
	baseline: 'GtSecurityMsr';
	load
```

## Load Lepiter

After installing with Metacello, you will be able to execute

```
#BaselineOfGtSecurityMsr asClass loadLepiter
```
